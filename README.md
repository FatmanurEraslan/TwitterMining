# TwitterMining

This project allows us to getting  a twitter trending topic data for 7 cities in turkey.
## Requirements

This project requires `Python 3.9.0` or newer run-time.

## Install


```sh
$ git clone  https://github.com/FatmanurEraslan/TwitterMining
$ cd TwitterMining
$ pip install tweepy
```
